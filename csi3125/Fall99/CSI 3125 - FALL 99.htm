CSI 3125 / FALL 99

CSI 3125 CONCEPTS OF PROGRAMMING LANGUAGES
(3 hours of lecture per week, 2 hours of lab per week, 4 credits)

Classification of programming languages. Syntactic description of programming languages. Imperative programming languages: data objects, data types, control structures, sub-programs, principles of implementation. Logic programming languages. Functional programming languages. Object-oriented programming languages. Modules. Prerequisites: CSI 2114, two programming language labs selected among CSI 2165, (CSI 2171 or CSI 2172 or CSI 2173) and CSI 2174. Cannot be combined for credit with CSI 2115.
