Program Execution
nThe loader brings into main memory only a few pieces of the process’ logical address space (including its starting point) nEach page/segment table entry has a present bit that is set only if the corresponding piece is in main memory nThe resident set  is the portion of the process’ address space that is in main memory nAn interrupt (memory fault or paging fault) is generated when a memory reference is on a page/segment not present in main memory
