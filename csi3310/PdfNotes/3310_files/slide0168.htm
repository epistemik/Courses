Algorithm 3 (Peterson’s algorithm)
nInitialization: flag[0]:=flag[1]:=false turn:= 0 or 1 nWillingness to enter CS specified by flag[i]:=true nIf both processes attempt to enter their CS simultaneously, only one turn value will last
Process Pi:
repeat
  flag[i]:=true;
    // I want in
  turn:=j;
   // but I let the other in
  while
   (flag[j]&turn=j){};
     CS
  flag[i]:=false;
   // I no longer want in
     RS
forever
