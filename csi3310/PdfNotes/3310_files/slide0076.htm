Symmetric Multiprocessing (SMP)
nA computer with multiple processors, where all processors have the same functionalities (symmetric) nThe OS schedules processes/threads across all the processors, which then become resources just as channels, etc. nExistence of multiple processors is transparent to the user.
nIncremental growth: just add another CPU!
nRobustness: a single CPU failure does not halt the system, only the performance is reduced.
