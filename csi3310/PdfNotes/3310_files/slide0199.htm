P/C: unbounded buffer
nRemarks:
ucheck that producer can run arbitrarily faster than consumer
uPutting signal(N) inside the CS of the producer (instead of outside) has no effect since the consumer must always wait for both semaphores before proceeding
uThe consumer must perform wait(N) before wait(S), otherwise deadlock occurs if consumer enters CS while the buffer is empty
udisaster will occur if one forgets to do a signal after a wait, eg. if a producer has an unending loop inside a critical section.
