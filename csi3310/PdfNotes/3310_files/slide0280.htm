Logical address used in simple segmentation with dynamic partitioning
nA CPU register holds the starting address of the the segment table of the program which the CPU executes.
nPresented with a logical address (segment number, offset) = (n,m), the CPU indexes (with n) the segment table to obtain the starting physical address k and the length l of that segment
nThe physical address is obtained by adding m to k 
uthe hardware also compares the offset m with the length l of that segment to determine if the address is valid
