Using xchg for mutual exclusion
nShared variable lock is initialized to 0 nEach Pi has a local variable key nThe only Pi that can enter CS is the one who finds lock=0 nThis Pi excludes all the other Pj by setting lock to 1
Process Pi:
repeat
  key:=1 
  repeat                     exchg(key,lock)
  until key=0;
     CS
  lock:=0;
     RS
forever
