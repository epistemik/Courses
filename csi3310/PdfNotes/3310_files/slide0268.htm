Buddy System
nWe start with the entire block of size 2^{U}
nWhen a request of size S is made:
uIf 2^{U-1} < S <= 2^{U} then allocate the entire block of size 2^{U}
uElse, split this block into two buddies, each of size 2^{U-1}
uIf 2^{U-2} < S <= 2^{U-1} then allocate one of the 2 buddies
uOtherwise one of the 2 buddies is split again
nThis process is repeated until the smallest block greater or equal to S is generated
nTwo buddies are coalesced whenever both of them become free
