When to Switch a Process ?
nA process switch may occur whenever the OS has gained control of CPU. ie when:
uSupervisor Call
Fexplicit request by the program (ex: I/O). The process will probably be blocked
uTrap
FAn error resulted from the last instruction. It may cause the process to be moved to exit state
uInterrupt
Fthe cause is external to the execution of the current instruction. Control is transferred to IH
