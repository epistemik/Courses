Address Translation
nRelative address is the most frequent type of logical address used in pgm modules
nSuch modules are loaded in main memory with all memory references in relative form
nIn the old times, the ‘loader’ program replaced relative addresses with physical ones at load time.
nNowadays physical addresses are calculated “on the fly” as the instructions are executed
nFor adequate performance, the translation from relative to physical address must by done by hardware
nThis translation is disabled when the system must work with physical addresses.
