Characteristics of Paging and Segmentation
nMemory references are dynamically translated into physical addresses at run time
ua program may be swapped in and out of main  memory such that it occupies different regions
nA program may be broken up into pieces (pages or segments) that do not need to be located contiguously in main memory
nHence: it becomes possible to execute a program that is not entirely loaded in memory.
ucomputation may proceed for some time if a sufficiently large portion of the program is in main memory (recall locality of reference)
