Combined Segmentation and Paging
nTo combine their advantages some processors and OS page the segments.
nSeveral combinations exists. Here is a simple one
nEach process has:
uone segment table
uone page table per segment
nThe virtual address consist of:
ua segment number: used to index the segment table whose entry gives the starting address of the page table for that segment
ua page number: used to index that page table to obtain the corresponding frame number
uan offset: used to locate the word within the frame
nSegment and page tables can themselves be paged!
