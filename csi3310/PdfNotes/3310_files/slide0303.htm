Paging
nEach page table entry contains a present bit to indicate whether the page is in main memory or not.
uIf it is in main memory, the entry contains the frame number of the corresponding page in main memory
uIf it is not in main memory, the entry may contain the address of that page on secondary memory (directly or indirectly).
nTypically, each process has its own page table
