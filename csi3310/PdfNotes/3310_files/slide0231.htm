Unix Pipes
nA shared bounded FIFO queue written by one process and read by another
ubased on the producer/consumer model
uOS enforces Mutual Exclusion: only one process at a time can access the pipe
uif there is not enough room to write, the producer is blocked, else he writes
uconsumer is blocked if attempting to read more bytes that are currently in the pipe
uaccessed by a file descriptor, like an ordinary file
uprocesses sharing the pipe are unaware of each other’s existence
