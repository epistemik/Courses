The Working Set Strategy
nthe working set concept suggest the following strategy to determine the resident set size
uMonitor the working set for each process
uPeriodically remove from the resident set of a process those pages that are not in the working set
uWhen the resident set of a process is smaller than its working set, allocate more frames to it
FIf not enough free frames are available, suspend the process (until more frames are available)
•I.e.: a process may execute only if its working set is in main memory
