Fetch Policy
nDetermines when a page should be brought into main memory. Two common policies:
uDemand paging only brings pages into main memory when a reference is made to a location on the page (ie: paging on demand only)
Fmany page faults when process first started but should decrease as more pages are brought in
uPrepaging brings in more pages than needed
Flocality of reference suggests that it is more efficient to bring in pages that are contiguous in virtual memory
Fefficiency not definitely established: the extra pages brought in are “often” not referenced
