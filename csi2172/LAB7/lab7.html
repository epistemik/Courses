       |
       +- ilist.h [.h]
       |
       +- ilist.cpp [.cpp]
       |
       +- ilist_inst.h [.h]
       |
       +- test.cpp [.cpp]

       (Makefile ilist.make)
    The interesting feature of this implementation is that a pointer is always kept on the last accessed element, hence sequential traversal can be done in linear time. This problem was also given as an assignment.
