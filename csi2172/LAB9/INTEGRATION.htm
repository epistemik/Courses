INTEGRATION

Integration

Suppose you and your partner have finished and debugged your code for stream A and B (the independent bonuses -- precedence of expressions and filling clipped figures -- are irrelevant). Here comes the big bonus. Wherever in the figure scripts you used a numeric constant for project B, you can replace it with a constant expression. Examples: 
  define canvas(30,30); 
  define canvas(20+5+5,2*15);
  set pen_color [200,0,50]; 
  set pen_color [300-100,5-5,(2+3)*(6+4)]
  ...

HINT

Wherever you had a lex.read_numeric(is) in the parser, you should have something like expr_parser.read(is)->evaluate().

Unless you have some debugging to do, this should do the trick and earns you a 3 marks reward!

HOW DO WE HAND IT IN?

  • create a subdirectory on the diskette, called integ. you will have all your code under this subdirectory that should make this program work including code for streams A and B.
  • printout of only those files that needed changes, and use a highlighter to show what had to be changed for the integration to work
