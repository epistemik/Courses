COORDSYS CLASS

MAPPING AND ROTATION OF COORDINATES

An instance of coordsys can be used to translate coordinates from one coordinate system to the other, and to return the resulting coordinate of a coordinate rotated.

ANNOTATED API DOCUMENTATION

    CLASS: coordsys 
         coordsys S(-20,-50,100,80);
     // S is a coordinate system with bottom left
     // corner <-20,-50> and upper right corner <100,80>
    METHODS:

      coordinate map(coordsys S, coordinate c)

        returns the image of coordinate c from system S in this system. 
          coordsys S1(-200,-200,100,0),
           S2(1.2,1.7,3.4,4.8);
  coordinate c1(60,82.5);

  coordinate c2 = S2.map(S1,c1);
  // c2 is the image in S2 of c1 defined in S1
      coordinate rotate(coordinate p, coordinate o, double a)

        returns the coordinate obtained by rotating p about o by a many degrees.
