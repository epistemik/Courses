Programming Exercise 1:

Write an ASM program that reads a text file from the standard input and displays on the screen the number of words contained in that file. For simplicity, assume that a word is a sequence of one or more alphabetic characters (hence, any sequence of one or more non alphabetic characters would terminate a word). As usual, use the getch macro to read the standard input and use the "<" symbol on the Win32 console command line to redirect the input from a file. I suggest that you use the putint macro to display the number of words. For example, suppose that we have a text file, named "my_file",  that contains these 2 lines:

Hi! is it cool
out there ? :-)

Here is a sample execution (user input is in bold):

C:\ASM>ass3-1 < my_file
This file contains 6 words
 

Programming Exercise 2:

A palindrome is a character string that reads the same forward or backward. In deciding if a string is a palindrome, we ignore blanks, punctuation, and letter case. For example "Madam, I'm Adam" or "A man, a plan, a canal, Panama!". Write a program that prompts the user to enter a string and decides wether or not it is palindrome.

Sample execution (user input is in bold):

C:\ASM>ass3-2
Please enter a string
Madam, I'm Adam!
This string is a palindrome

C:\ASM>ass3-2
Please enter a string
Madam, I'm Wilfrid!
This string is not a palindrome

Programming Exercise 3:

A character string STRING1 preceeds alphabetically another string STRING2 if and only if (a) the first character of STRING1 comes before the first character of STRING2 alphabetically, or (b) the first N-1 characters of the strings are identical, but the Nth character of STRING1 preceeds the Nth character of STRING2, or (c) STRING1 matches the beginning of STRING2, but STRING2 is longer.

Write a program that lets the user enter two character strings and decides which string comes first alphabetically, or if the strings are identical.

Sample execution (user input is in bold):

C:\ASM>ass3-3
Enter a first string
abcd
Enter a second string
ae
The first string preceeds alphabetically the second string

C:\ASM>ass3-3
Enter a first string
abcd
Enter a second string
ab
The second string preceeds alphabetically the first string

C:\ASM>ass3-3
Enter a first string
abcd
Enter a second string
abcd
The strings are identical
