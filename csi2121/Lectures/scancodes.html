Scan Codes

Each key on a IBM and compatible keyboard has a scan code. All scan codes values below are  in hexadecimal. When a key is pressed, the keyboard sends a make code to port 60h. When a key is released a break code is sent. The values below show the make codes. A break code can be obtained  from the make code by ORing it with 80h.
